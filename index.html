Latest Projects

Pokemon App

The Pokedex project is a web application that allows users to browse and search for information on Pokemon. Upon entering the site, the user is prompted to enter their name. Once entered, the user can browse a list of all Pokemon available in the API or filter them by type. Additionally, the user can search for a specific Pokemon by name. The project is built using React, Redux, and React Router Dom to manage state, routing, and data retrieval from the API.

Rick and Morty Location Search Tool

This project is a React-based web application that allows users to search for locations featured in the popular TV show "Rick and Morty". Users can search for locations by their ID number, which are displayed in the show's episodes. The app retrieves data from the Rick and Morty API and displays it in an easy-to-use interface

Top Scorer App

This is a React-based web app that shows the top scorers of the five biggest European soccer leagues (Premier League, La Liga, Serie A, Bundesliga, and Ligue 1), as well as the UEFA Champions League and Europa League, since the 2015 season. It uses the API-Football service to retrieve data on the top scorers and presents it in an easy-to-use interface. Users can select the league and season to view the top scorers.

Wheather App

This is a React-based weather app that uses a weather API to automatically retrieve the weather information for your current location based on your coordinates. The app also features a search input button that allows you to look up weather information for other cities. With a user-friendly interface and accurate and up-to-date weather data, this app provides a convenient way to stay informed about the weather wherever you are.

E-commerce

My E-commerce project is a web application that uses HTML, CSS, and JavaScript to create an online store. The site uses LocalStorage to save changes made by the user, and an API to retrieve product data. With a streamlined interface and easy navigation, customers can easily browse and purchase products with just a few clicks. Overall, this E-commerce project provides a convenient and hassle-free online shopping experience for users.
