Últimos Proyectos

Pokemon App

El proyecto Pokedex es una aplicación web que permite a los usuarios buscar información sobre Pokemon. Tras ingresar el nombre, se puede buscar entre una lista completa de Pokemon disponibles en la API o filtrarlos por tipo. Además, se puede buscar un Pokemon en específico por su nombre. Este proyecto está construido con React, Redux y React Router Dom para gestionar el estado, enrutamiento y datos de la API.

Rick and Morty

Este proyecto es una aplicación web basada en React que permite a los usuarios buscar ubicaciones presentadas en la popular serie de televisión "Rick y Morty". Los usuarios pueden buscar ubicaciones por su número de identificación, que se muestran en los episodios del programa. La aplicación recupera datos de la API de Rick y Morty y los muestra en una interfaz fácil de usar.

Top Scorer App

Esta app web de React muestra los máximos goleadores de las 5 mayores ligas de fútbol europeas, incluyendo la UEFA Champions League y Europa League, desde el 2015. Usa API-Football para obtener los datos y los presenta en una interfaz amigable. Los usuarios pueden filtrar por liga y temporada para ver los goleadores.

App del Tiempo

Esta es una app meteorológica en React que utiliza una API para obtener datos meteorológicos de la ubicación actual del usuario o de otras ciudades mediante una función de búsqueda. Ofrece una interfaz amigable y datos precisos y actualizados para mantener al usuario informado sobre el clima.

E-commerce

Mi proyecto de e-commerce es una tienda en línea creada con HTML, CSS y JavaScript. Utiliza LocalStorage para guardar cambios del usuario y una API para recuperar datos de productos. Con una interfaz sencilla y fácil navegación, los clientes pueden comprar productos con solo unos clics. Ofrece una experiencia de compra en línea conveniente y sin problemas.
